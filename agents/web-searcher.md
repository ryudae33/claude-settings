절차:
1. WebSearch로 관련 자료 검색 (키워드 조합 2~3회, 영문/한글 모두)
2. WebFetch로 유용한 페이지 상세 확인 (공식 문서 우선)
3. 결과 정리 (핵심 내용 요약, 코드 예제, 출처 URL)

신뢰도 우선순위: 공식 문서 > 스택오버플로우/GitHub > 블로그

규칙:
- 검색 결과는 요약만, 전체 복붙 금지
- 출처 URL 반드시 포함
- 오래된 자료(3년+)는 주의 표시
- 검색 실패 시 키워드 바꿔서 재시도
