# Panasonic MINAS A6 서보 드라이버 - 펄스 제어 가이드

## 1. 펄스 제어 기본 설정

### 1.1 위치 제어 모드 개요

MINAS A6 시리즈는 고성능 다기능 타입(SF)과 일반 위치 제어 타입(SE) 등으로 구분됩니다. 위치 제어 모드는 상위 컨트롤러의 펄스 지령에 따라 23bit(8,388,608 분해능) 엔코더의 정밀도를 극대화하여 구동됩니다.

- **포토커플러(Photocoupler) 입력:** 노이즈 절연이 필요한 환경에 필수적이며, 라인 드라이버 및 오픈 컬렉터 출력을 가진 상위 제어기에 대응합니다.
- **라인 리시버(Line Receiver) 입력:** 차동 신호 방식을 채택하여 고주파수(최대 8 Mpulse/s) 펄스 입력이 가능하며 고속 제어에 최적화되어 있습니다.
- **시스템 유형 주의:** 세이프티 기능이 필요한 경우 다기능 타입(SF)을 선정해야 하며, 일반 위치 제어 타입(SE)은 기능 안전 규격(STO 등)에 대응하지 않습니다.

### 1.2 펄스열 입력 방식 및 배선 가이드

| 입력 방식 | 대응 인터페이스 | 특징 및 배선 주의사항 |
| :--- | :--- | :--- |
| **포토커플러 입력** | 라인 드라이버 / 오픈 컬렉터 | 회로 간 전기적 절연을 통해 외부 노이즈 전도를 차단 |
| **라인 리시버 입력** | 라인 드라이버 | 고주파수 특성이 우수하며, 반드시 차동 신호 배선 준수 |

> **참고:** 오픈 컬렉터 인터페이스 사용 시에는 신호의 상승/하강 속도 한계로 인해 최대 입력 주파수가 포토커플러(라인 드라이버 기준 500 kpulse/s) 사양보다 낮아질 수 있습니다. 고속 운전이 필요한 경우 반드시 라인 드라이버 방식을 권장합니다.

### 1.3 입력 펄스 형태 (Pr0.07)

1. **정방향/부방향 (Sign + Pulse):** 펄스열로 속도와 거리를, 부호 신호(High/Low)로 회전 방향을 제어
2. **A상/B상 (A/B Phase):** 두 상의 위상차(90도)를 통해 방향을 판단하며, 4배수 처리를 통해 분해능 활용
3. **지령/방향 (CW/CCW):** 정회전 펄스(CW)와 역회전 펄스(CCW)가 독립된 라인으로 입력

### 1.4 최대 입력 펄스 주파수

- **포토커플러 입력 사용 시:** 최대 **500 kpulse/s** (라인 드라이버 출력 기준)
- **라인 리시버 입력(A/B상 방식) 사용 시:** 최대 **8 Mpulse/s** (차동 입력 기준)

### 1.5 전자기어비 설정 및 계산식

전자기어비는 상위 지령 단위와 모터 회전량을 일치시키는 핵심 설정입니다. 1회전당 8,388,608 펄스(23bit 엔코더)가 기준입니다.

> **중요:** Pr0.08(1회전당 펄스 수) 설정이 '0'이 아닐 경우, Pr0.09(분자)와 Pr0.10(분모) 설정은 완전히 무시됩니다. 분자/분모 형태의 정밀 설정이 필요하다면 반드시 Pr0.08을 '0'으로 설정하십시오.

**전자기어 계산식:**

```
위치 지령 처리 = 입력 지령 펄스 x (분자(Pr0.09) / 분모(Pr0.10))
```

- **예시:** 상위 지령 10,000 펄스당 모터 1회전 → 분자/분모 = 8,388,608 / 10,000
- **허용 범위:** 1/1000배 ~ 8000배

### 1.6 위치 제어 관련 핵심 파라미터

| 파라미터 번호 | 명칭 | 설정값 범위 및 의미 |
| :--- | :--- | :--- |
| **Pr0.05** | 지령 펄스 입력 선택 | 0: 포토커플러 입력, 1: 라인 리시버 입력 |
| **Pr0.06** | 지령 펄스 극성 설정 | 0~1: 지령 펄스의 논리 극성 및 방향 전환 |
| **Pr0.07** | 지령 펄스 입력 모드 설정 | 0: Sign+Pulse, 1: A/B상, 2: CW/CCW |
| **Pr0.08** | 모터 1회전당 지령 펄스 수 | 0: Pr0.09/0.10 활성, 1~8388608: 해당 값으로 1회전 |
| **Pr0.09** | 전자 기어 분자 | Pr0.08=0일 때 유효 (범위: 1 ~ 2^30) |
| **Pr0.10** | 전자 기어 분모 | Pr0.08=0일 때 유효 (범위: 1 ~ 2^30) |

### 1.7 커넥터 X4 핀 배치

- **펄스 지령 입력 (Photocoupler):**
  - 3번(PULSH1), 4번(PULSL1): 펄스 입력
  - 5번(SIGNH1), 6번(SIGNL1): 부호 입력
- **펄스 지령 입력 (Line Receiver):**
  - 44번(PULSH2), 45번(PULSL2): 펄스 입력
  - 46번(SIGNH2), 47번(SIGNL2): 부호 입력
- **엔코더 피드백 펄스 출력:**
  - 21번(OA+), 22번(OA-): A상 출력 (라인 드라이버)
  - 48번(OB+), 49번(OB-): B상 출력 (라인 드라이버)
  - 23번(OZ+), 24번(OZ-): Z상 출력 (라인 드라이버)
  - 25번(OZ): Z상 출력 (오픈 컬렉터)

### 1.8 배선 주의사항

- **D종 접지:** 감전 방지 및 노이즈 억제를 위해 100 Ohm 이하의 접지를 반드시 시행
- **전원 분리:** 모터 브레이크용 DC24V와 제어 전원용 DC24V는 반드시 절연된 별도의 전원 사용
- **페라이트 코어:** 전원선 및 모터선에 설치, 케이블 시스(외피)를 제거한 후 L1, L2, L3(또는 U, V, W) 전선을 일괄 코어에 권선
- **환경:** 오염도 2(Pollution Degree 2) 환경 기준, IP54 이상의 밀폐 제어반 설치 권장

## 2. 제어 신호 및 입출력

### 2.1 필수 입력 신호 (X4 커넥터)

- **SON (서보 온):** 서보 모터의 통전 및 구동 가능 상태를 제어
- **PULS/SIGN (지령 펄스):** 상위 지령에 따른 위치 정보 입력, 파라미터로 입력 형태 선택
- **CLR (편차 카운터 클리어):** 드라이버 내부의 위치 편차 카운터 값을 강제로 0으로 리셋
- **INH (지령 펄스 금지):** 활성화된 동안 드라이버는 입력 지령 펄스를 카운트하지 않고 무시

입력 신호 할당: 10개의 범용 입력을 제공하며, **Pr4.00~Pr4.09** 파라미터로 기능을 자유롭게 할당 가능

| 입력 인터페이스 | 대응 신호 형태 | 최대 지령 펄스 주파수 |
| :--- | :--- | :--- |
| **포토커플러 입력** | 라인 드라이버 / 오픈 컬렉터 | 500 kpulse/s |
| **라인 리시버 입력** | 라인 드라이버 전용 (A/B상 등) | 8 Mpulse/s |

### 2.2 출력 신호 (X4 커넥터)

- **ALM (알람):** 보호 기능 작동 시 출력. **37번(ALM+)** 및 **36번(ALM-)** 핀에 고정 할당
- **RDY (서보 준비 완료):** 주회로 전원 투입 및 에러가 없는 상태에서 SON 입력이 가능한 준비 상태
- **INP (위치결정 완료):** 위치 편차가 설정된 범위 내로 수렴했음을 표시

6개의 범용 출력은 **Pr4.10~Pr4.15** 파라미터를 통해 논리 및 기능 선택 가능

### 2.3 위치결정완료(INP) 판정 조건

지령 펄스 입력이 종료되고, 드라이버 내부의 '위치 편차'가 사용자 설정 범위 내로 들어오면 INP 신호가 출력됩니다.

- **Pr4.02 (위치결정 완료 범위):** INP 신호를 출력하기 위한 위치 편차의 한계값 설정
- **그룹 4 (I/F 설정):** 출력 신호의 기능 할당 및 논리 설정
- **그룹 0 (기본 설정):** 제어 모드 및 전자 기어비 등 동작 환경 정의

### 2.4 원점 복귀(Homing)

- **앱솔루트 모드:** 멀티턴 데이터를 유지하므로 전원 투입 시마다 별도의 원점 복귀가 불필요
- **인크리멘탈 모드:** 전원 투입 시 반드시 Z상 탐색을 포함한 원점 복귀 시퀀스를 수행하여 기계적 원점 확립 필요

### 2.5 비상정지 및 알람 리셋 시퀀스

**알람 발생 시 시퀀스:**
1. 드라이버 내부 보호 기능에 의한 ALM 신호 출력
2. 상위 제어 회로에서 ALM 신호를 수신하여 주회로 전자 접촉기(MC) OFF
3. 주회로 전원 공급 중단 및 모터 정지

> **주의:** '즉시 정지' 기능은 전원이 공급된 상태에서의 제어 감속 정지입니다. 알람 신호와 동시에 주회로 전원(MC)을 즉시 차단할 경우, 감속에 필요한 동력을 잃게 되어 강제적으로 다이나믹 브레이크(DB) 정지로 전환됩니다.

**리셋:** 발생 원인을 제거하고 장비의 안전을 확보한 후, 리셋 신호를 입력하거나 전원을 재투입

### 2.6 타이밍 차트

**전원 투입 및 서보 온 시퀀스:**
1. 제어 전원(L1C, L2C) 투입 → CPU 초기화 시작
2. 주회로 전원(L1, L2, L3) 투입 → 동력 공급 대기
3. SON 신호 입력 → 내부 릴레이 및 구동 회로 활성화
4. 서보 락(Servo Lock) → 내부 릴레이 시퀀스 완료 후 모터가 구동력 유지 상태 진입

**다이나믹 브레이크(DB):**
- 알람 발생 또는 서보 오프 시 설정된 정지 방식에 따라 DB 작동
- 비상 정지용이므로 통상적인 정지 수단으로 반복 사용 금지
- 고속 운전 중 DB 작동 후 약 10분 정도 냉각 시간 필요

## 3. 파라미터 및 튜닝

### 3.1 제어 모드 선택 (Pr0.01)

| 설정값 | 제어 모드 명칭 | 주요 특징 |
| :--- | :--- | :--- |
| **0** | 위치 제어 | 상위 컨트롤러의 펄스 열 지령에 의한 위치 결정 전용 모드 |
| **3** | 위치 / 속도 제어 | I/O 신호를 통해 위치 제어와 속도 제어를 실시간 전환 |
| **4** | 위치 / 토크 제어 | I/O 신호를 통해 위치 제어와 토크 제어를 실시간 전환 |
| **6** | 풀 클로즈 제어 | 외부 스케일 피드백을 직접 받아 기계 끝단의 위치를 정밀 제어 |

### 3.2 전자기어 관련 파라미터

- **Pr0.08 (지령 펄스 분주 분자):** 0 설정 시 엔코더 분해능(2^23 = 8,388,608)으로 자동 적용 (범위: 0 ~ 2^30)
- **Pr0.09 (지령 펄스 분주 분모):** 초기값 10000 (범위: 1 ~ 2^30)
- **Pr0.10 (제2 지령 펄스 분주 분자):** 외부 입력 신호로 기어비 전환 시 사용 (범위: 0 ~ 2^30)
- **허용 범위:** 1/1000 ~ 8000배 (초과 시 지령 펄스 분주 체배 이상 보호 작동)

### 3.3 게인 관련 파라미터

A6 시리즈는 **2자유도(2-DOF) 제어 방식**을 채택하여 지령 추종성과 외란 억제력을 독립적으로 조정 가능합니다.

- **Pr1.00 (제1 위치 루프 게인):** 위치 지령에 대한 추종 응답성. 초기값 480 (단위: 0.1/s, 범위: 0~30000)
- **Pr1.01 (제1 속도 루프 게인):** 속도 루프의 응답성, 시스템 강성과 직결. 초기값 270 (단위: 0.1Hz, 범위: 1~32767)
- **Pr1.02 (제1 속도 루프 적분 시간 상수):** 속도 제어의 정상 상태 편차 제거. 초기값 210 (단위: 0.1ms, 범위: 1~10000)

### 3.4 오토튜닝

- **Pr0.02 (실시간 오토튜닝 설정):** 실제 운전 중 부하 관성비를 실시간 추정하여 최적 게인 자동 갱신. 초기값 1 (범위: 0~6)
- **Pr0.03 (기계 강성 설정):** 오토튜닝 시 기준 강성. 초기값 13 (범위: 0~31). 높을수록 응답성 증가하나 진동 가능성 있으므로 점진적으로 상향

### 3.5 가감속 필터 파라미터

- **Pr2.22 (지령 평활 필터):** 위치 지령에 1차 지연 필터 적용. 초기값 0 (단위: 0.1ms, 범위: 0~32767)
- **Pr2.23 (지령 FIR 필터):** 이동 평균 필터로 급격한 변화 억제. 초기값 0 (단위: 0.1ms, 범위: 0~10000)

### 3.6 위치결정완료 범위

- **Pr4.31 (위치결정완료 범위):** INP 신호 출력 편차 범위. 초기값 10 (엔코더 단위, 범위: 0~2^23)

### 3.7 토크 제한

- **Pr0.13 (제1 토크 제한):** 정격 토크 대비 최대 출력 제한. 초기값 500 (단위: %, 범위: 0~500)
- **위치 제어 타입(SE):** 파라미터(Pr0.13, Pr5.22 등)를 통해 디지털 방식으로 토크 제한
- **다기능 타입(SF):** 파라미터 방식 외 X4 아날로그 전압 입력으로 실시간 토크 제한 가변 가능

### 3.8 핵심 파라미터 종합 일람표

| 번호 | 명칭 | 초기값 | 범위 | 설명 |
| :--- | :--- | :--- | :--- | :--- |
| **Pr0.01** | 제어 모드 선택 | 0 | 0~6 | 0:위치, 1:속도, 2:토크, 3:위치/속도, 4:위치/토크, 6:풀클로즈 |
| **Pr0.02** | 실시간 오토튜닝 | 1 | 0~6 | 부하 관성 실시간 추정 → 게인 자동 설정 |

### 3.9 안전 주의사항

- 게인 조정: 시스템 안정성 확인 없이 극단적 변경 금지 (진동/폭주 위험)
- 시운전: 반드시 모터를 기구부와 분리한 상태에서 단독 동작 확인 후 체결
- 잔류 전압: 전원 차단 후 최소 15분 이상 대기, 충전 램프(CHARGE) 소등 확인 후 작업
- 환경: 물, 기름, 절삭유, 부식성/인화성 가스 환경에서 사용 금지
