6.1.1. Web Client

및 DIP 4는 꺼짐(OFF)으로 설정)을 권장합니다. (자세한 내용은 Ethernet 인터페이스 설정 장
을 참조하십시오.)
그리고 다음 단계를 따르십시오.
•
제공된 UTP 케이블로 컴퓨터에 Compute Box를 연결하십시오.
•
제공된 전원 공급 장치로 Compute Box의 전원을 켜십시오.
•
Compute Box의 LED가 파란색에서 녹색으로 바뀌는 데 1분 정도 걸립니다.
•
Compute Box의 IP 주소를 찾으려면 OnRobot 디스커버리 도구를 여십시오.
•
컴퓨터에서 웹 브라우저를 열고 Compute Box의 IP 주소를 입력하십시오 (공장 초기값은
192.168.1.1).
로그인 페이지가 다음과 같이 열립니다.
공장 초기 관리자 로그인 정보는 다음과 같습니다.
사용자 이름: admin
비밀번호: OnRobot
초기 로그인을 위해 새로운 비밀번호를 입력해야 합니다. (비밀번호는 최소 8자 이상이어야
