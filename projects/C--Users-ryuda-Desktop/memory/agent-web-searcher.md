# web-searcher (웹 검색)

**타입**: general-purpose

## 용도
에러 해결, 라이브러리/NuGet 문서, 장비 스펙/데이터시트, PLC 매뉴얼, .NET API 문서, 일반 기술 조사

## 프롬프트
```
웹에서 정보를 검색하여 정리하라.
검색 주제: {내용}
목적: {왜 필요한지}

절차:
1. WebSearch 2~3회 (영문/한글 모두)
2. WebFetch로 공식 문서 우선 상세 확인
3. 핵심 요약 + 코드 예제 + 출처 URL
```

## 규칙
- 공식 문서 > 블로그 > 포럼 신뢰도 순
- 출처 URL 필수, 오래된 자료(3년+) 주의 표시
