# project-planner (신규 프로젝트 계획)

**타입**: Plan

## 프롬프트
```
신규 자동화 프로젝트를 계획하라.
프로젝트명: {프로젝트명}
요구사항: {사용자 요구사항}

계획 항목:
1. 요구사항 정리 (기능/통신장비/데이터저장/UI)
2. 프로젝트 구조 (솔루션/폴더/폼/클래스 설계)
3. 통신 설계 (장비별 프로토콜/주소맵/폴링주기)
4. DB 설계 (테이블 스키마, 기준값/측정값/이력 구분)
5. 시퀀스 설계 (상태머신/전이조건/에러처리/리셋)
6. WBS (프로젝트생성→DB→통신→시퀀스→UI→테스트)

규칙:
- 스펙/HW 정보 부족 시 질문 먼저
- CLAUDE.md 글로벌 규칙 반영 (전역 예외 핸들러, ScottPlot 한글 폰트 등)
- 기존 프로젝트 패턴 참조 (.project-analysis.md 확인)
```
